##  Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/lnemenl/CPP.git
   cd CPP
   ```

2. **Build & Run:**
     ```bash
     cd CPP00 - CPP09/Exercise
     make
     ./ModuleExecutable
     ```
